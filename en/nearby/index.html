Nearby Communication

Trace hands off notes only between devices that are physically nearby. It is a quiet, proximity-based synchronization model.

1. Communication Technologies

Trace uses Bluetooth and Local Network (AWDL) for direct device-to-device communication.

2. Purpose

The purpose is to synchronize Trace data directly between users in the same place.

3. Typical Range

The practical communication range is typically within several tens of meters, depending on conditions.

4. No External Server

Synchronization is fully peer-to-peer and never routed through external servers.

5. Collected Data

No personal information is collected. Only diff-based Trace packet synchronization is exchanged.
